## 🌟 Project Overview

A stylish **Jewelry E-commerce Website** built using HTML & CSS featuring a clean UI, wishlist, and cart interface.
Designed to provide a smooth and elegant online shopping experience 💖

---

## 🚀 Features

✨ Animated product hover effects
🛒 Add to Cart button
❤️ Wishlist functionality (UI)
💰 Click to view price interaction
📱 Fully Responsive Design
🎨 Elegant white & peach themed layout
